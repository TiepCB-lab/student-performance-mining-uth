# Backend

Backend FastAPI cho du an Student Performance Mining UTH.

## Chay API

```powershell
cd backend
python run.py
```

API mac dinh chay tai `http://127.0.0.1:8000`.

## Endpoint chinh

- `GET /`: form HTML nhap thong tin sinh vien.
- `GET /schema`: danh sach truong dau vao cua model.
- `POST /predict`: du doan ket qua tu JSON.
